# xargs 命令

xargs命令用于将上一个命令通过管道输出的内容, 作为命令行参数传递给下一个命令

xargs后面的命令默认是echo

即: stdin -> **argv

# tee 命令

tee命令用于将标准输入复制多份, 分别输出到不同的文件, 并输出到标准输出中去

可以用来复制文件

* 输出管道中某一段的内容:
```sh
... | tee >(xargs echo) >/dev/null | ...
```

* 不退出vim使用sudo保存文件

命令`:w !{cmd}`表示把当前缓冲区的内容传给后面的命令

```sh
:w !sudo tee % 
```

# echo 命令

echo命令用于将命令行参数输出到标准输出

即: stdin -> stdout
