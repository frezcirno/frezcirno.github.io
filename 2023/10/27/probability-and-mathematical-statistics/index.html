#基本概念

随机试验
可以在相同的条件下重复进行，并且每次试验的结果不确定，但试验前可以明确试验的所有可能结果。
样本空间
随机试验EEE 的所有可能结果组成的集合，记为SSS。
样本点
样本空间中的元素，记为ω\omegaω。
事件
样本空间SSS 的子集称为随机事件，简称事件，通常用大写字母A,B,C,...A, B, C, ...A,B,C,... 表示。
基本事件
只包含一个样本点的随机事件。
  • 必然事件: 包含所有样本点的随机事件。
  • 不可能事件: 不包含任何样本点的随机事件。

事件关系和运算

  • 包含关系:A⊂BA \subset BA⊂B，AAA 包含于BBB。
  • 相等关系:A=BA = BA=B，AAA 等于BBB。
  • 和事件:A∪BA \cup BA∪B，AAA 与BBB 至少有一个发生。
  • 积事件:A∩BA \cap BA∩B，AAA 与BBB 同时发生。
  • 差事件:A−BA - BA−B，AAA 发生而BBB 不发生。
  • 互斥事件:A∩B=∅A \cap B = \emptysetA∩B=∅，AAA 与BBB 不可能同时发生。
  • 逆事件/对立事件:A∪B=SA \cup B = SA∪B=S，AAA 与BBB 至少有一个发生。
频数
在nnn 次试验中，事件AAA 发生的次数，记为nAn_AnA​。
频率
事件AAA 发生的频率，记为fn(A)=nAnf_n(A) = \frac{n_A}{n}fn​(A)=nnA​​。
概率
事件AAA 发生的可能性大小，记为P(A)P(A)P(A)，满足以下三个条件:
  • 非负性:P(A)≥0P(A) \geq 0P(A)≥0。
  • 规范性:P(S)=1P(S) = 1P(S)=1。
  • 可列可加性: 若A1,A2,...A_1, A_2, ...A1​,A2​,... 两两互斥，则P(⋃i=1∞Ai)=∑i=1∞P(Ai)P(\bigcup_{i=1}^{\infty}A_i) = \sum_{i=1}^{\infty}P(A_i)P(⋃i=1∞​Ai​)=∑i=1∞​P(Ai​)。

概率的性质推论：

  • P(∅)=0P(\emptyset) = 0P(∅)=0。
  • 有限可加性: 若A1,A2,...A_1, A_2, ...A1​,A2​,... 两两互斥，则P(⋃i=1nAi)=∑i=1nP(Ai)P(\bigcup_{i=1}^{n}A_i) = \sum_{i=1}^{n}P(A_i)P(⋃i=1n​Ai​)=∑i=1n​P(Ai​)。
  • 包含事件的概率: 若A⊂BA \subset BA⊂B，则P(A)≤P(B)P(A) \leq P(B)P(A)≤P(B)。
  • P(A)≤1P(A) \leq 1P(A)≤1。
  • 互补事件的概率:P(Aˉ)=1−P(A)P(\bar{A}) = 1 - P(A)P(Aˉ)=1−P(A)。
  • 加法公式:P(A∪B)=P(A)+P(B)−P(A∩B)P(A \cup B) = P(A) + P(B) - P(A \cap B)P(A∪B)=P(A)+P(B)−P(A∩B)。
古典概型/等可能概型
符合以下条件的概率模型称为古典概型:
  1. 试验的样本空间是有限的。
  2. 试验的每个基本事件发生的可能性相同。
